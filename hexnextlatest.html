Chat Loader V2

Overview

This version of the Chat Loader introduces style configurations from the front-end, allowing clients to dynamically update animations, colors, and gradients without code changes. It uses GSAP and Framer Motion libraries to provide smooth and customizable animations (bounce, fade, etc.).

  • Client-facing style customization
  • Color and gradient configuration
  • Animation type selection (bounce, fade, etc.)
  • GSAP and Framer Motion integration
  • Google Sheets data fetching (if applicable)
